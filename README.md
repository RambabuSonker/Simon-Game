# Simon Game 🎮

A fun and interactive memory game built using **HTML, CSS, and JavaScript**. The game challenges players to remember and repeat an increasingly long sequence of colors.

## 🚀 Features
- Random color sequence generation
- User input validation
- Increasing difficulty with each level
- Sound effects and animations
- Game over screen with restart functionality

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- jQuery

## 🎯 How to Play
1. Press any key to start the game.
2. Watch the color pattern shown by the game.
3. Repeat the sequence by clicking the colored buttons in the correct order.
4. Each successful round adds one more color to the sequence.
5. The game ends if you click the wrong button. Press any key to restart.
